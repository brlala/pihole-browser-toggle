Pi-hole Toggle — screenshot composer

Chrome Web Store screenshots must be 1280×800 or 640×400 (JPEG / 24-bit PNG, no alpha). The popup is only 304 px wide, so this tool centers a real screenshot of the popup or options page on a brand-colored canvas of the correct size and adds an optional caption.

How to capture the popup

  1. In Chrome, click the extension's toolbar icon to open the popup.
  2. Right-click inside the popup → Inspect. DevTools opens for the popup document.
  3. In DevTools, press Ctrl+Shift+P → type "Capture full size screenshot" → Enter. A PNG of the popup downloads.
  4. Repeat for both states (blocking / disabled) and for the options page (right-click the icon → Options, then DevTools + capture).
  5. Drop each PNG into this tool and download the composed 1280×800 version.
